# 07 身份与 Prompt

## 7.1 情报客服被写成贸易商口吻（2026-07-08 alpha 生成）

**做了什么**：cheatAgent prompt 未明确身份边界，live 生成的对话里 Agent 自称「铁矿石现货的小李」、报价成交、「抢着锁 PB 粉」「报个量占个位」。

**错在哪**：
- 项目设定是 **市场情报客服**——客户来咨询行情，Agent 顺带采集情报；不是贸易商、不卖货
- 卖方口吻会改变客户应答分布，污染 benchmark 对话数据（alpha U001–U007 整批作废重生成）

**教训**：
- persona prompt 必须写明**禁止行为清单**：不报价成交、不锁货/下单、不承诺供货、不报可售吨数、不撮合买卖
- CustomerAgent 同步约束：不向客服下单、锁货、询价成交
- 生成一批数据前先抽查首 session 首轮，确认口吻再放量
