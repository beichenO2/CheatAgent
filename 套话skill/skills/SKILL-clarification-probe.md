# clarification-probe — 澄清提问

## 何时使用

- `inferred_gaps` 含未明 region/grade/volume
- 开场 1–3 轮，客户只说「铁矿」「什么价」「有没有货」
- `resistance_level` ≤ medium，对方愿意配合
- **不用**：已有完整采购清单；用户拒答细节

## 输出

```json
{ "utterance": "...", "target_slots": ["grade","port","volume"], "phase": "PROBE" }
```

## 话术模板

1. **品类**：「您这次主要看进口粉矿现货，还是港口块矿，还是远期？」
2. **品位**：「常见 62% 普氏对标、61.5% 低铝、65% 高品，您对标哪档？」
3. **港口**：「青岛、日照、唐山港您更关心哪个港？」
4. **数量节奏**：「大概什么量级、什么时候要？」
5. **用途**：「高炉配矿还是贸易转手？我好按场景给您报。」

## 执行要点

- 每轮只澄清 1–2 个 slot；澄清后给一句市场反馈维持互惠
- 问题选项须与 `price_snapshot`/港存统计一致，禁止编造货源

## 禁忌

- 连续 4 轮只问不答（违反互惠）
- 未确认品类就追问敏感经营数据
