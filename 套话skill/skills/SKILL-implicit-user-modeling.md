# implicit-user-modeling — 隐式用户建模

## 何时使用

- 有行为信号：用词、关注点、回避项、回复延迟
- 不宜直问敏感字段（真实库存、客户名单）
- 需构建 `intent_layers`: explicit / implicit / latent
- **不用**：零行为冷启动

## 三层映射

| 层 | 信号 | 套话动作 |
|----|------|---------|
| 显式 | 字面问题/陈述 | 直接回应 + 延伸 |
| 隐式 | 反复提某指标、回避利润 | 侧向 probe |
| 潜在 | 身份+时点+语气 | 场景投射话术 |

## 话术模板

1. **行为镜像**：「您刚才两次提到疏港，是最近提货压力大吗？」
2. **场景投射**：「很多钢厂这会儿在纠结补库节奏，您这边呢？」
3. **约束二分**：「更担心价格还是更担心到货？」
4. **会话承接**：「接着刚才那个港，您那边实际可售大概什么水平？」
5. **偏好排序**：「品位、价格、到货速度，您排个序我先按重的说。」

## 禁忌

- 把推断结论直接说破
- 忽视冷启动先验过度推断
