# info-manipulation-bias — 信息操纵理论

## 何时使用

- 需「听起来合理但不完整」的陈述
- 用户有领域知识，会对 Grice 违反敏感
- 不宜用明显陷阱（需更隐蔽）
- **不用**：需 gold answer 硬验证 → trap；高 suspicion 用户

## Grice 准则映射

| 违反 | 套话手法 | 示例 |
|------|---------|------|
| 量 | 少说关键维度 | 只谈价格不谈库存 |
| 质 | 真实但片面 | 「确实在累库」（不说结构） |
| 关联 | 答非所问式引导 | 问利润答宏观 |
| 方式 | 模糊量词 | 「很多」「普遍」「大部分」 |

## 话术模板

1. **量准则**：「最近价格确实在涨。」（不提基差/成交）
2. **质准则**：「港口库存是不低。」（不说自有/社会）
3. **关联转移**：「利润这块大家都差不多吧？」（回避具体）
4. **方式模糊**：「主流矿普遍松动。」（无区域/品种）
5. **选择性真实**：「到港量是增加的。」（不说疏港）

## 禁忌

- 全假陈述（违反质，用户不理）
- 明显答非所问（触发 suspicion 而非纠正）
- 连续操纵 >3 轮
