# reactance-biased-statement — 心理反抗 / 有偏陈述

## 何时使用

- `resistance_level` = low/medium
- 有 partial claim 但缺量化细节
- 不宜直接追问敏感数字
- **不用**：resistance 高 → RECOVER；用户无专业认知 → trap/socratic

## 有偏陈述四要素

1. 非全假（像业内传闻） 2. 方向性偏差 3. 可验证锚点 4. 低控制语言（「听说」「有人讲」）

## 话术模板

1. **库存亏损偏**：「听说港口累库厉害，压货的多数在亏，你们也这样吗？」
2. **涨价预期偏**：「矿价还要大涨，现在卖货的都亏了，您那边呢？」
3. **区域以偏概全**：「北方钢厂都在减产，需求很差？」
4. **时间窗口偏**：「上周那波涨价已经结束了？」
5. **成本偏**：「进口到岸成本普遍高于现货？」
6. **物流偏**：「听说某港要限装，货都出不去？」

## 提取目标

从反驳中提取：库存量级、盈亏、区域差异、成交节奏、套保情况。

## 禁忌

- 「你必须承认…」等高控制措辞
- resistance 高时继续 CHALLENGE
- 完全离谱谣言（用户会忽略）
